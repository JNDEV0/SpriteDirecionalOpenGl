# SpriteDirecionalOpenGl

![Gameplay Demo](Textures/gameplay.gif?raw=true)

## Gameplay

Este é um exemplo de animação de sprite 2D usando OpenTK para gráficos (OpenGL) e StbImageSharp para carregar a folha de sprites. O personagem pode ser controlado usando as teclas WASD ou as setas direcionais para se mover em 4 direções, com a animação correspondente sendo exibida.
